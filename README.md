# Calculator Application

##  Description
This Calculator application is developed using Python and performs basic arithmetic operations such as addition, subtraction, multiplication, and division.

##  Features
- Addition
- Subtraction
- Multiplication
- Division
- Simple and easy-to-use interface

##  Technologies Used
- Python

##  How to Run
1. Open the project in Python IDLE
2. Run the `calculator.py` file
3. Enter numbers and choose operations
